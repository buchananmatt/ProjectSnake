\doxysubsection{Detailed Description}
\+: provides the interface between the \doxylink{classbocan_1_1snake_1_1_game}{Game} and the user. prints game elements to the screen and receives keyboard input from the user. 

\+: acts as the interface with the ncurses library. responsible for reading and writing information to the savefile. \begin{DoxyRefDesc}{Todo}
